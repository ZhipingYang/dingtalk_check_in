> 安卓手机远程打卡羡慕，iOSer苦逼啊
> 
> 原理：手机放在公司运行该软件彻夜不关闭😂😂😂，直到上午定时启动钉钉，因为手机在公司，则在自动打卡。

### 要求
- iOS 7.0 以上
- 一部手机在公司
- 运行软件的手机上安装并登录好钉钉
- 公司在钉钉平台上开启极速打卡功能（早上公司附近打开钉钉就可以打卡）
- 授权允许接受定时推送
- 授权允许打开钉钉，理由如下 ↓

> 先直接点击“直接打开钉钉”按钮，第一次打开需要授权，以后就可以直接打开了。然后就可以自动打卡了。。。
